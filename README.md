# Paralel Computing 🤖

<h3>Definisi</h3>
<p>Paralel Computing merupakan teknik untuk menjalankan beberapa tugas secara bersamaan dengan cara dibagi menjadi beberapa unit. Menggunakan jalur komunikasi anter memory untuk saling membagi tugas.
secara singkatnya paralel computing menggunakan berbagai prosessor dalam satu sistem.</p>

<ul>
  ➕<b>Kelebihan</b>
  <li>Computasi yang lebih cepat</li>
  <li>Penggunaan CPU yang lebih Efficient</li>
  <li>Mengurangi Overhed</li>
  <li>Meningkatkan Kapasistas Operations</li>

  ➖<b>Kekurangan</b>
  <li>Harga Hardware yang Mahal</li>
  <li>Jika Satu Terkendala, Akan Berdampak Pada Pemrosesan Yang Lain</li>
  <li>Sinkronasi Tugas yang Komples</li>
</ul>
